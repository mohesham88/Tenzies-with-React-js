# Tenzies-with-React-js
## Objective of the game:
Make all ten dices show the same number.
## Technologies
```
  React + javascript
```
## Set up 
install npm
```
  npm install
```
second install react confetti and react use lib
```
  npm install react-use
  npm install react-confetti
```
